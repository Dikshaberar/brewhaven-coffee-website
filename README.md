# ☕ BrewHaven Coffee Website

A modern and responsive coffee shop website built using **HTML, CSS, and JavaScript**.
This project showcases a stylish UI design for a coffee brand with smooth animations and interactive elements.

---

## 🚀 Features

* ✨ Responsive design
* 🎯 Smooth scrolling navigation
* 🛍️ Product showcase section
* 🎨 Clean and modern UI
* ⚡ Hover animations for products

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

---

## 📂 Project Structure

```
├── index.html
├── style.css
├── script.js
└── images/
```

---

## ▶️ How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
